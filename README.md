# 💰 消费透视镜 (Consumer Telescope)

> 财富管理的行为层基础设施 —— 帮你"看懂"消费，而非仅仅记录

## 项目定位

**记账的"第二眼"** —— 钱迹帮你记，消费透视镜帮你"看懂"。

- **透视感**：一眼看出"钱花哪去了"，不是数字是洞察
- **对照感**：有参照系，知道自己"偏没偏"
- **行动感**：不只是展示，是告诉你"可以怎么改"

## 核心功能

- 快速记账（3秒内完成一笔记账）
- AI 消费结构分析（找出"可优化空间"）
- 省钱启示录（匿名同类对比）
- 周账单分享（生成好看周报图）

## 技术架构

| 模块 | 方案 |
|------|------|
| 记账入口 | Android 原生（Kotlin + Jetpack Compose）|
| 数据存储 | SQLite（Room 抽象）|
| AI 分析 | 本地推理小模型 / 外部 API |
| 跨平台同步 | 可选、用户发起的 |

## 资源评估

- **原型阶段**：~0 成本（现有设备 + Claude Code）
- **产品化阶段**：350~1800元/年
- **推广阶段**：7300~30000元/年
