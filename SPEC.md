# 消费透视镜 (Consumer Telescope) - 项目规格说明书

> V1 聚焦版 —— 纯工具定位

---

## 1. 项目概述

**项目名称**：消费透视镜
**项目类型**：Android 原生应用
**一句话描述**：像钱迹一样方便快捷的记账工具，加上 AI 消费结构透视能力。

---

## 2. 核心功能

- **快速记账**：3秒内完成一笔记账，极简流程
- **消费结构透视**：AI 分析消费结构，找出"可优化空间"
- **本地优先存储**：SQLite + 文件备份，隐私第一

---

## 3. 技术架构

| 层级 | 技术选型 |
|------|----------|
| **前端** | Android Native（Kotlin + Jetpack Compose） |
| **本地数据库** | SQLite（通过 Room 抽象） |
| **AI 能力** | 本地推理小模型 / 外部 API（MiniMax/DeepSeek/阿里百炼） |
| **网络** | Retrofit + OkHttp |
| **依赖注入** | Hilt |
| **异步处理** | Kotlin Coroutines + Flow |
| **架构模式** | MVVM + Clean Architecture |

---

## 4. V1 MVP 功能列表

### 4.1 消费记录模块
- 添加消费记录（金额、分类、日期、备注）
- 消费分类管理（餐饮、购物、交通、娱乐等默认分类）
- 消费记录列表展示与搜索
- 编辑/删除消费记录

### 4.2 数据存储模块
- Room 数据库设计与实现
- 消费记录表、分类表
- 本地数据持久化

### 4.3 AI 透视模块
- 调用 AI API 分析消费结构
- 生成"可优化空间"洞察
- 给出具体省钱建议

### 4.4 基础 UI
- 主页面：消费记录列表
- 添加记录页面
- 透视结果展示页面
- 设置页面（AI API 配置）

### 4.5 数据获取（V1）
- ✅ 手动记账
- ❌ 其他平台导入（后续版本支持）

---

## 5. 非功能需求

- 最低支持 Android 7.0（API 24）
- 离线优先，数据本地存储
- 用户隐私数据不上传第三方服务器
