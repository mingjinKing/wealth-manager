# 消费透视镜 (Consumer Telescope) - 项目规格说明书

## 1. 项目概述

**项目名称**：消费透视镜
**项目类型**：Android 原生应用
**一句话描述**：一款帮助用户理性消费、分析消费行为的智能助手应用，通过 AI 技术提供消费洞察与建议。

---

## 2. 核心功能

- **消费记录管理**：快速记录日常支出，支持分类、金额、备注
- **AI 消费分析**：基于 OpenAI API 分析消费结构，识别浪费点
- **消费数据可视化**：图表展示支出趋势与分类占比
- **智能建议**：AI 提供个性化省钱建议与预算规划

---

## 3. 技术架构

| 层级 | 技术选型 |
|------|----------|
| **前端** | Android Native（Kotlin + Jetpack Compose） |
| **本地数据库** | SQLite（通过 Room 抽象） |
| **AI 能力** | OpenAI API（GPT-4o mini） |
| **网络** | Retrofit + OkHttp |
| **依赖注入** | Hilt |
| **异步处理** | Kotlin Coroutines + Flow |
| **架构模式** | MVVM + Clean Architecture |

---

## 4. 第一阶段 MVP 功能列表

### 4.1 消费记录模块
- 添加消费记录（金额、分类、日期、备注）
- 消费分类管理（餐饮、购物、交通、娱乐等默认分类）
- 消费记录列表展示与搜索
- 编辑/删除消费记录

### 4.2 数据存储模块
- Room 数据库设计与实现
- 消费记录表、分类表
- 本地数据持久化

### 4.3 AI 分析模块（最小可用）
- 调用 OpenAI API 分析单条消费记录
- 简单的消费建议生成
- API Key 配置与管理

### 4.4 基础 UI
- 主页面：消费记录列表
- 添加记录页面
- AI 分析结果展示页面
- 设置页面（API Key 配置）

---

## 5. 非功能需求

- 最低支持 Android 7.0（API 24）
- 离线优先，数据本地存储
- 用户隐私数据不上传第三方服务器
