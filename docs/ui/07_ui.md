## 二、首页 · 消费仪表盘（Dashboard）

### 2.1 屏幕类型
**单页滚动式仪表盘**（Single-page Scrollable Dashboard）

### 2.2 核心组件

| 组件名称 | 组件类型 | 数据内容 |
|----------|----------|----------|
| 顶部导航栏 | TopAppBar | 标题"消费透视镜"、设置入口 |
| 周概览卡片 | SummaryCard | 本周消费总额、环比上周涨跌 |
| 消费结构列表 | ProgressList | 分类名 + 进度条 + 金额 |
| AI 建议卡片 | InsightCard | AI 生成的省钱建议文案（最多3条）|
| 哇时刻预览卡片 | RewardPreviewCard | 本周节省金额预览、触发状态标识 |
| 悬浮添加按钮 | FAB (Floating Action Button) | 快速记账入口（BottomNav 替代后保留）|

### 2.3 布局结构

```
┌──────────────────────────────────────┐
│ [设置⚙️]    💰 消费透视镜      [更多⋮] │  ← 顶部导航栏
│                                      │
│  ┌────────────────────────────────┐ │
│  │ 本周总计        vs 上周        │ │  ← 周概览卡片
│  │ ¥ 2,019        ↑12%           │ │
│  └────────────────────────────────┘ │
│  ← 12dp 间距                        │
│  ┌────────────────────────────────┐ │
│  │ 消费结构                   [▶]  │ │
│  │ 餐饮  ████████████░░░  ¥1,240  │ │  ← 进度条（语义色）
│  │ 购物  ██████░░░░░░░░  ¥480    │ │
│  │ 交通  ████░░░░░░░░░░  ¥210    │ │
│  └────────────────────────────────┘ │
│  ← 12dp 间距                        │
│  ┌────────────────────────────────┐ │
│  │ 💡 AI 优化建议                   │ │
│  │ "外卖每周减少1次，月可节省约¥200" │ │
│  └────────────────────────────────┘ │
│  ← 12dp 间距                        │
│  ┌────────────────────────────────┐ │
│  │ 🎉 哇时刻                [已触发]│ │
│  │ 本周少花 ¥237.50               │ │
│  └────────────────────────────────┘ │
└──────────────────────────────────────┘
                              [ ＋ ]     ← FAB（右下悬浮）
┌──────────────────────────────────────┐
│  🏠 首页   │  ➕ 添加  │  💡 透视  │  🏆 成就 │  ← BottomNav 4栏
└──────────────────────────────────────┘
```

**布局节奏**：
- 顶部：轻量导航，占屏幕高度 ≤ 8%
- 上部：周概览卡片，卡片圆角 12dp
- 中部：消费结构列表（核心内容区），占屏幕高度 40-50%
- 下部：AI 建议 + 哇时刻，卡片堆叠
- 底部：BottomNav 固定，FAB 叠加在 BottomNav 上方右侧

**⚠️ 设计决策**：
- Dashboard 分隔：纯 12dp 间距，无分隔线
- 卡片圆角：12dp（与深色主题适配）
- BottomNav 存在但 FAB 仍保留作为快捷入口

### 2.4 风格氛围

| 维度 | 规范 |
|------|------|
| **色彩语义** | 暖色系（橙/红）= 超支/警示，冷色系（蓝/绿）= 节约/正常，金色（#ffc880）= 成就/哇时刻 |
| **进度条语义色** | 当周该分类消费 > 基准 → #E55B5B（暖红）；≤ 基准 → #4A90D9（冷蓝）|
| **视觉层次** | 数据卡片为主，强调"一眼看到重点"，非数据信息克制展示 |
| **情感基调** | 正向激励型：强调"攒到了"而非"花多了" |
| **字体** | Manrope（设计系统字体，非 Roboto）|
| **字重层级** | 金额数字：Manrope Bold 32-48sp；分类标签：Medium 14sp；辅助文案：Regular 12sp |
| **留白规范** | 卡片间距 12dp，内边距 16dp，与屏幕边缘间距 16dp |

## 三、记账页 · 3秒极速录入（Quick Add）

### 3.1 屏幕类型
**全屏沉浸式数字输入页**（Full-screen Immersive Number Input）

### 3.2 核心组件

| 组件名称 | 组件类型 | 行为 |
|----------|----------|------|
| 金额展示区 | AmountDisplay | 默认聚焦数字键盘，实时显示输入金额 |
| 分类选择器 | HorizontalChipGroup | 横向滚动分类标签（餐饮/购物/交通/娱乐/居住/医疗/学习/其他）|
| 数字键盘 | NumericKeypad | 0-9 + 小数点 + 删除 + 确认 |
| 日期选择器 | DateChip | 默认"今天"，点击展开日期选择 |

### 3.3 布局结构

```
┌──────────────────────────────────────┐
│ [←]                         [今天]  │
│                                      │
│              ¥ 0.00                  │
│           （默认36sp灰色）            │
│                                      │
│  [餐饮] [购物] [交通] [娱乐]         │
│  [居住] [医疗] [学习] [其他]         │
│                                      │
│   ┌───┬───┬───┬───┐               │
│   │ 1 │ 2 │ 3 │ ← │               │
│   ├───┼───┼───┼───┤               │
│   │ 4 │ 5 │ 6 │   │               │
│   ├───┼───┼───┼───┤               │
│   │ 7 │ 8 │ 9 │ ✓ │               │
│   ├───┼───┼───┼───┤               │
│   │ · │ 0 │ 00│   │               │
│   └───┴───┴───┴───┘               │
│                                      │
└──────────────────────────────────────┘
┌──────────────────────────────────────┐
│  🏠 首页   │  ➕ 添加  │  💡 透视  │  🏆 成就 │
└──────────────────────────────────────┘
```

### 3.4 风格氛围

| 维度 | 规范 |
|------|------|
| **交互原则** | 零认知负担：打开即输入，无需二次聚焦 |
| **金额默认态** | 占位符 ¥ 0.00：36sp，灰色；进入输入态：48sp，白色 |
| **分类选中态** | 选中项：实心填充（#ffc880）+ 白色文字；未选中：描边 + 灰色文字 |
| **键盘按键** | 圆角矩形 8dp，按下态有 8% 透明度提升 |
| **确认按钮** | 主色（#ffc880）填充，与数字键区分 |
| **删除按钮** | 图标按钮，右侧独立列 |
| **字体** | Manrope |
| **分类 Chip 滚动提示** | 两端渐变遮罩（Gradient Fade）|

## 七、全局动效规范（Motion Spec）

### 7.1 全局过渡动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 页面切换（BottomNav 切换）| 300ms | ease-in-out | 标准 Android 页面过渡 |
| 卡片进入 | 200ms | ease-out | Dashboard 列表项依次出现 |
| 模态弹窗 | 250ms | ease-out | 从底部滑入 |

### 7.2 FAB 动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| FAB 出现 | 150ms | ease-out | 页面加载完成后淡入 |
| FAB 按下 | 100ms | ease-in | 缩放至 0.95 |

### 7.3 记账页（Quick Add）动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 页面进入 | 200ms | ease-out | 从底部滑入 |
| 金额数字变化 | 即时 | — | 无动画，保证输入响应 |
| 分类 Chip 选中 | 100ms | ease-out | 背景色切换 |
| 确认成功 | 150ms | ease-out | 整体淡出 + 向右滑出 |

### 7.4 哇时刻页动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 页面进入 | 400ms | ease-out | 全屏淡入 |
| 成就图标 🎉 | 300ms | spring | 缩放弹跳（0 → 1.1 → 1.0）|
| 金额翻转 | 600ms | ease-out | 数字滚动（roll）效果 |
| 等价物列表 | 100ms × n | ease-out | 依次出现，stagger 间隔 |
| 渐变分隔线 | 400ms | ease-out | 从左至右扫过 |
| 成就徽章 | 300ms | ease-out | 淡入 + 微缩放（0.8 → 1.0）|

### 7.5 消费结构进度条动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 进度条填充 | 400ms | ease-out | 从 0% 动画填充至目标值 |
| 语义色切换 | 200ms | ease-in-out | 超支/节约状态变化时颜色渐变过渡 |

### 7.6 动效关键词规范

| Keyword | 具体参数 |
|---------|----------|
| **ease-out** | cubic-bezier(0.0, 0.0, 0.2, 1.0) |
| **ease-in** | cubic-bezier(0.4, 0.0, 1.0, 1.0) |
| **ease-in-out** | cubic-bezier(0.4, 0.0, 0.2, 1.0) |
| **spring** | damping: 0.7，stiffness: 300 |
| **数字滚动 roll** | 数字从大到小翻转，类 ATM 取款机效果 |

## 九、页面流转（v1.2 更新）

```
App启动
    ↓

    ↓
首页（仪表盘）
    ↓ [FAB ＋] 或 BottomNav [添加]

    ↓ [BottomNav 成就]

    ↓

    ↓

    ↓ [返回]
成就页
```
