## 二、首页 · 消费仪表盘（Dashboard）

### 2.1 屏幕类型
**单页滚动式仪表盘**（Single-page Scrollable Dashboard）

### 2.2 核心组件

| 组件名称 | 组件类型 | 数据内容 |
|----------|----------|----------|
| 顶部导航栏 | TopAppBar | 标题"消费透视镜"、设置入口 |
| 周概览卡片 | SummaryCard | 本周消费总额、环比上周涨跌 |
| 消费结构列表 | ProgressList | 分类名 + 进度条 + 金额（餐饮/购物/交通/娱乐等）|
| AI 建议卡片 | InsightCard | AI 生成的省钱建议文案 |
| 哇时刻预览卡片 | RewardPreviewCard | 本周节省金额预览、触发状态标识 |
| 悬浮添加按钮 | FAB (Floating Action Button) | 快速记账入口 |

### 2.3 布局结构

```
┌──────────────────────────────────────┐
│ [←]  💰 消费透视镜            [⚙️] │  ← 顶部导航栏
│                                      │
│  ┌────────────────────────────────┐ │
│  │ 本周总计        vs 上周        │ │  ← 周概览卡片（16dp圆角）
│  │ ¥ 2,019        ↑12%           │ │
│  └────────────────────────────────┘ │
│  ← 12dp 间距（纯间距，无分隔线）    │
│  ┌────────────────────────────────┐ │
│  │ 消费结构                   [▶] │ │  ← Section Header
│  │ 餐饮  ████████████████░░ ¥1,240│ │  ← 进度条（语义色）
│  │ 购物  ██████████░░░░░░░ ¥480  │ │
│  │ 交通  ██████░░░░░░░░░░ ¥210  │ │
│  │ 娱乐  ███░░░░░░░░░░░░░  ¥89   │ │
│  └────────────────────────────────┘ │
│  ← 12dp 间距                        │
│  ┌────────────────────────────────┐ │
│  │ 💡 AI 优化建议                   │ │  ← AI 建议卡片
│  │ "外卖每周减少1次，月可节省约¥200" │ │
│  │                         [查看详情]│ │
│  └────────────────────────────────┘ │
│  ← 12dp 间距                        │
│  ┌────────────────────────────────┐ │
│  │ 🎉 哇时刻                [已触发]│ │  ← 哇时刻卡片
│  │ 本周少花 ¥237.50               │ │
│  │ "查看我的成就 →"               │ │
│  └────────────────────────────────┘ │
│                              [ ＋ ]  │  ← FAB（右下，8dp elevation）
└──────────────────────────────────────┘
```

**布局节奏**：
- 顶部：轻量导航，占屏幕高度 ≤ 8%
- 上部：周概览卡片，卡片圆角 16dp
- 中部：消费结构列表（核心内容区），占屏幕高度 40-50%
- 下部：AI 建议 + 哇时刻，卡片堆叠
- 底部：FAB 右下角悬浮，距边缘 16dp
- **卡片分隔：纯 12dp 间距，无分隔线**

**⚠️ 设计决策：Dashboard 分隔方式 = 纯间距**

### 2.4 风格氛围

| 维度 | 规范 |
|------|------|
| **色彩语义** | 暖色系（橙/红）= 超支/警示，冷色系（蓝/绿）= 节约/正常，金色 = 成就/哇时刻 |
| **进度条语义色** | 当周该分类消费 > 周均基准 → #E55B5B（暖红）；≤ 基准 → #4A90D9（冷蓝）；进度条颜色随语义动态变化 |
| **视觉层次** | 数据卡片为主，强调"一眼看到重点"，非数据信息克制展示 |
| **情感基调** | 正向激励型：强调"攒到了"而非"花多了" |
| **字体层级** | 金额数字：Roboto Bold 32-48sp；分类标签：Medium 14sp；辅助文案：Regular 12sp |
| **留白规范** | 卡片间距 12dp，内边距 16dp，与屏幕边缘间距 16dp |

**⚠️ 设计决策：进度条色彩 = 超支/节约语义变色**（非分类固定色）

## 三、记账页 · 3秒极速录入（Quick Add）

### 3.1 屏幕类型
**全屏沉浸式数字输入页**（Full-screen Immersive Number Input）

### 3.2 核心组件

| 组件名称 | 组件类型 | 行为 |
|----------|----------|------|
| 金额展示区 | AmountDisplay | 默认聚焦数字键盘，实时显示输入金额 |
| 分类选择器 | HorizontalChipGroup | 横向滚动分类标签（餐饮/购物/交通/娱乐/居住/医疗/学习/其他）|
| 数字键盘 | NumericKeypad | 0-9 + 小数点 + 删除 + 确认 |
| 日期选择器 | DateChip | 默认"今天"，点击展开日期选择 |

### 3.3 布局结构

```
┌──────────────────────────────────────┐
│ [←]                         [今天]  │  ← 返回 + 日期
│                                      │
│              ¥ 0.00                  │  ← 金额展示
│           （默认36sp灰色）            │     输入态：48sp白色
│                                      │
│  [餐饮] [购物] [交通] [娱乐]         │  ← 分类选择（单选，选中态）
│  [居住] [医疗] [学习] [其他]         │  ← 横向滚动，两端渐变遮罩
│                                      │
│   ┌───┬───┬───┬───┐               │
│   │ 1 │ 2 │ 3 │ ← │               │  ← 数字键盘
│   ├───┼───┼───┼───┤               │
│   │ 4 │ 5 │ 6 │   │               │
│   ├───┼───┼───┼───┤               │
│   │ 7 │ 8 │ 9 │ ✓ │  ← 确认/保存  │
│   ├───┼───┼───┼───┤               │
│   │ · │ 0 │ 00│   │               │
│   └───┴───┴───┴───┘               │
│                                      │
└──────────────────────────────────────┘
```

**布局节奏**：
- 上方：金额展示区，占据屏幕 25-30%，居中对齐
- 中部：分类选择器，高度 56dp，固定不动；两端渐变遮罩提示可滚动
- 下方：数字键盘，占据屏幕 45-50%，按键尺寸统一 64dp × 56dp

### 3.4 风格氛围

| 维度 | 规范 |
|------|------|
| **交互原则** | 零认知负担：打开即输入，无需二次聚焦 |
| **金额默认态** | 占位符 ¥ 0.00：36sp，灰色 #8E8E93；进入输入态：48sp，白色 #FFFFFF |
| **分类选中态** | 选中项：实心填充 + 白色文字；未选中：描边 + 灰色文字 |
| **键盘按键** | 圆角矩形 8dp，按下态有 8% 透明度提升 |
| **确认按钮** | 主色（深墨 #1A1A2E）填充，与数字键区分 |
| **删除按钮** | 图标按钮，无文字，右侧独立列 |

## 四、哇时刻页 · 成就感冲击（Reward Celebration）

### 4.1 屏幕类型
**全屏单一成就页**（Full-screen Single Achievement View）

### 4.2 核心组件

| 组件名称 | 组件类型 | 内容 |
|----------|----------|------|
| 成就图标区 | CelebrationIcon | 🎉 emoji，居中，高度 80dp |
| 主文案区 | HeadlineText | "本周你少花了"，22sp |
| 金额展示区 | HeroAmount | 节省金额，超大金色，64sp |
| 渐变分隔线 | GradientDivider | 金色→透明渐变，2dp |
| 原因描述区 | CauseText | "因为你少点了5次外卖，节省了 ¥240" |
| 等价物展示 | EquivalentList | "相当于：一顿双人火锅 / 半件优衣库..." |
| 成就徽章 | AchievementBadge | 徽章图标 + 成就名称（简化版，无卡片包裹）|
| 进度暗示 | ProgressHint | "连续4周达标解锁 → 攒钱达人" |
| 行动按钮区 | ActionButtons | "分享本周成绩"（主按钮）、"查看透视 →"（次按钮）|

### 4.3 布局结构

```
┌──────────────────────────────────────┐
│ [←]                                 │
│                                      │
│                 🎉                   │
│                                      │
│            本周你少花了               │
│                                      │
│           ¥ 237.50                  │  ← 金色 64sp，带微阴影
│                                      │
│  ════════════════════════════════  │  ← 金色→透明渐变分隔线
│                                      │
│      因为你少点了5次外卖              │
│         节省了 ¥240                  │
│                                      │
│           相当于：                    │
│      • 一顿双人火锅                  │  ← 等价物精简至 1-2 项
│                                      │
│      🏆 精打细算                     │  ← 简化徽章（无卡片包裹）
│                                      │
│     连续4周达标解锁 → 攒钱达人       │  ← 次级文字
│                                      │
│       [ 分享本周成绩 ]               │
│       [ 查看消费透视 → ]             │
│                                      │
└──────────────────────────────────────┘
```

**布局节奏**：
- 上半屏：情感冲击区（图标 + 文案 + 金额），占 55%
- 中半屏：原因 + 等价物 + 成就（精简），占 30%
- 下半屏：行动按钮，占 15%

**⚠️ 设计决策：哇时刻页下半屏精简** — 等价物 ≤ 2 项，徽章去除卡片包裹，进度暗示降为次级文字

### 4.4 风格氛围

| 维度 | 规范 |
|------|------|
| **主色调** | 金色强调（#F5A623），背景深墨（#1A1A2E）|
| **情感强度** | 高：全屏只有一个核心数字，足够留白 |
| **金额处理** | Roboto Bold 64sp，金色填充，文字描边 + 微阴影 |
| **等价物文案** | 次级文字灰色，Bullet List 排列，精简至 1-2 项 |
| **成就徽章** | 简化版：图标 + 名称，无卡片包裹；金光晕：blur 8dp，opacity 40%，spread 4dp |
| **渐变分隔线** | 颜色：#F5A623 → transparent，方向：水平 |

## 五、通用设计规范

### 5.1 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色/背景 | #1A1A2E | 深墨色，沉浸感 |
| 次级背景 | #16213E | 卡片背景 |
| 强调/成就 | #F5A623 | 金色，省钱/成就/哇时刻 |
| 安全/节约 | #4A90D9 | 冷蓝，正常/节约（进度条 ≤ 基准时）|
| 警示/超支 | #E55B5B | 暖红，超支/异常（进度条 > 基准时）|
| 主文字 | #FFFFFF | 白色 |
| 次级文字 | #9E9EA4 | 灰色（**v1.1 提升对比度**，原 #8E8E93）|
| 分割线 | #2D2D44 | 淡分割 |

**⚠️ 设计决策：次级文字色 = #9E9EA4**（WCAG AA+ 对比度）

### 5.2 组件圆角

| 组件 | 圆角规范 |
|------|----------|
| 页面级卡片 | 16dp |
| 按钮/输入框 | 8dp |
| 标签/分类 Chip | 20dp（全圆角）|
| 数字键盘按键 | 8dp |

### 5.3 间距系统（8pt Grid）

| 用途 | 间距 |
|------|------|

### 5.4 阴影规范

## 六、全局动效规范（Motion Spec）

> **⚠️ v1.1 新增 | 基于设计评审决策 #3**

### 6.1 全局过渡动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 页面切换（Activity/Fragment 跳转）| 300ms | ease-in-out | 标准 Android 页面过渡 |
| 卡片进入 | 200ms | ease-out | Dashboard 列表项依次出现 |
| 模态弹窗 | 250ms | ease-out | 从底部滑入 |

### 6.2 FAB 动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| FAB 出现 | 150ms | ease-out | 页面加载完成后淡入 |
| FAB 按下 | 100ms | ease-in | 缩放至 0.95 |
| FAB 展开（Speed Dial）| 200ms | ease-out | 子选项依次展开（stagger 50ms）|

### 6.3 记账页（Quick Add）动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 页面进入 | 200ms | ease-out | 从底部滑入 |
| 金额数字变化 | 即时 | — | 无动画，保证输入响应 |
| 分类 Chip 选中 | 100ms | ease-out | 背景色切换 |
| 确认成功 | 150ms | ease-out | 整体淡出 + 向右滑出 |

### 6.4 哇时刻页（Reward Celebration）动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 页面进入 | 400ms | ease-out | 全屏淡入 |
| 成就图标 🎉 | 300ms | spring | 缩放弹跳（0 → 1.1 → 1.0）|
| 金额翻转 | 600ms | ease-out | 数字滚动（roll）效果 |
| 等价物列表 | 100ms × n | ease-out | 依次出现，stagger 间隔 |
| 渐变分隔线 | 400ms | ease-out | 从左至右扫过 |
| 成就徽章 | 300ms | ease-out | 淡入 + 微缩放（0.8 → 1.0）|

### 6.5 消费结构进度条动效

| 动效场景 | 时长 | 缓动曲线 | 说明 |
|----------|------|----------|------|
| 进度条填充 | 400ms | ease-out | 从 0% 动画填充至目标值 |
| 语义色切换 | 200ms | ease-in-out | 超支/节约状态变化时颜色渐变过渡 |

### 6.6 动效关键词规范

为确保设计意图在工程侧准确落地，定义以下 motion keywords：

| Keyword | 具体参数 |
|---------|----------|
| **ease-out** | cubic-bezier(0.0, 0.0, 0.2, 1.0) |
| **ease-in** | cubic-bezier(0.4, 0.0, 1.0, 1.0) |
| **ease-in-out** | cubic-bezier(0.4, 0.0, 0.2, 1.0) |
| **spring** | damping: 0.7，stiffness: 300 |
| **数字滚动 roll** | 数字从大到小翻转，类 ATM 取款机效果 |

## 七、页面流转

```
App启动
    ↓
首页（仪表盘）
    ↓ [FAB ＋]
记账页（3秒）
    ↓ [确认✓]
返回首页（刷新数据）
    ↓
每周结算触发
    ↓
哇时刻页
    ↓ [返回]
首页
```
