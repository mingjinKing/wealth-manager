## 四、组件清单

### 4.1 TopAppBar

| 元素 | 说明 |
|------|------|
| 返回按钮 | ← 返回上一页（侧滑导航） |
| 标题 | "怎么花" |
| 历史按钮 | 查看历史记录（右侧 IconButton） |

### 4.2 快捷入口卡片

| 状态 | 说明 |
|------|------|
| 默认 | 白色背景，阴影，图标+标题+描述 |
| 按下 | 背景变浅色 + 轻微缩放 |
| 选中 | 边框高亮（主题色） |

**卡片样式：**
```
背景：#FFFFFF（浅色）/ #1A1A2E（深色）
圆角：16dp
内边距：16dp
阴影：elevation 2dp
间距：12dp
```

### 4.3 消息气泡

| 类型 | 样式 |
|------|------|
| 用户消息 | 右对齐，主题色背景，白色文字，圆角 16dp（左侧） |
| AI 消息 | 左对齐，白色背景，深色文字，圆角 16dp（右侧），卡片样式 |

### 4.4 输入框

| 元素 | 说明 |
|------|------|
| 输入框 | 圆角 24dp，背景灰色，hint "说说你的想法..." |
| 发送按钮 | 主题色圆形按钮，右侧，Icon: Send |

### 4.5 AI 思考状态

```
┌─────────────────────────────────────┐
│  🤖 好的！帮你分析一下...            │
│  ⏳ 正在整合你的消费数据...           │  ← 流式输出
│  ⏳ 正在查询历史价格...              │
└─────────────────────────────────────┘
```

**特点：**
-逐字显示，有打字机效果
-状态文字用 ⏳ 图标
-加载动画：三个点闪烁

### 4.6 快捷操作栏

| 按钮 | 图标 | 说明 |
|------|------|------|
| 有用 | 👍 | 标记为有帮助 |
| 追问 | 📝 | 继续追问 |
| 收藏 | ⭐ | 收藏这条建议 |
| 分享 | 📤 | 分享给好友 |

## 十一、与透视页的关系

```
透视页（算算账）          怎么花
    ↓                      ↓
被动分析              主动决策
已消费数据            将消费决策
    ↓                      ↓
定期推送周报          随时问答
```

**共享组件：**
- AI 消息气泡样式（可复用 InsightsScreen）
- 打字机效果组件
- 用户画像获取（共享 AgentContext）

**差异化：**
- 快捷入口卡片（怎么花特有）
- 引导流程（怎么花特有）
- 历史记录（怎么花特有）
