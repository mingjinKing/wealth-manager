### 4.4 月度概览卡片（MonthOverviewCard）

- **比例**：4:3（aspectRatio = 4f / 3f）
- **背景**：可切换的背景图片（drawable 目录下的 `background_1`、`background_2`）
- **点击交互**：点击卡片区域切换背景图片
- **渐变蒙层**：黑色透明渐变（0.4f → 0 → 0.3f），确保文字在亮色背景上清晰
- **数据展示**：
  - 本月支出：大号金色数字（42sp Bold）
  - 本月收入：白色中号文字
  - 近7日支出：白色中号文字

### 4.5 每日账单卡片交互

- **点击单条消费条目** → 进入记账页编辑
- **长按单条消费条目** → 弹出删除确认对话框
- **编辑流程**：点击 → 修改金额/分类/备注/日期 → 确认 → 返回首页更新
- **删除流程**：长按 → 确认弹窗 → 确认删除 → 返回首页更新

### 4.6 空状态

当无记账记录时，显示：
- 主文案："暂无记账记录"
- 副文案："点击 + 开始记账"

## 五、记账页 · AddExpenseScreen

### 5.1 屏幕类型

**全屏沉浸式数字输入页**。

### 5.2 核心组件

| 组件名称 | 行为 |
|----------|------|
| 顶部导航 | 返回箭头 + 标题"添加账单"/"修改账单" + 日期选择器 |
| Tab 切换 | "添加账单" / "添加收入" 两个 Tab |
| 分类选择器 | 4列网格布局，emoji + 文字，选中态实心金色填充 |
| 音符输入 | BasicTextField，最多50字，与金额并排一行 |
| 金额显示 | 实时显示，数字键盘输入，支持 +/- 计算 |
| 数字键盘 | 5列布局：数字 + 功能键 |
| 日期选择器 | 右上角可点击，弹出 DatePickerDialog |

### 5.3 布局结构

```
┌──────────────────────────────────────┐
│ [←]    添加账单              [今天] │  ← 可点击选择日期
│ ─────────────────────────────────── │
│  [添加账单]        [添加收入]        │  ← Tab 切换
│ ─────────────────────────────────── │
│                                      │
│  [🍗餐饮] [🛒购物] [🚇交通] [🎮娱乐]│  ← 分类网格（4列）
│  [🏠居住] [💊医疗] [📚学习] [📦其他]│
│                                      │
│  ┌────────────────┐                  │
│  │ 备注            │    ¥ 0        │  ← 音符+金额并排
│  └────────────────┘                  │
│                                      │
│   1   2   3   +   −                │
│   4   5   6   ⌫   ✓                │  ← 数字键盘（5列）
│   7   8   9   ·                   │
│   0       0                   │
└──────────────────────────────────────┘
```

### 5.4 数字键盘布局

| 行 | 按键 |
|----|------|
| 1 | `7` `8` `9` `+` `−` |
| 2 | `4` `5` `6` `⌫` `✓` |
| 3 | `1` `2` `3` `·` |
| 4 | `0` (占2列宽度) |

### 5.5 计算器逻辑

- 支持 + / − 运算
- 输入数字 → 点击运算符 → 输入第二个数字 → 点击 ✓ 计算结果
- 显示实时预览（pendingValue + pendingOperator）

### 5.6 日期选择

- 默认显示"今天"，点击弹出 Material3 DatePickerDialog
- 日期显示格式：今天 / 昨天 / 4月2日
- 选定日期随账单一起存入数据库

### 5.7 分类长按删除

- 长按分类 → 弹出删除确认对话框
- 删除后自动取消选中状态

### 5.8 记账完成流程

1. 选择分类 → 输入金额 → ✓ 确认
2. 新建：自动跳转回首页（Dashboard）
3. 编辑：返回上一级（popBackStack）

## 十一、哇时刻触发公式

```
savedAmount = last_4_week_avg - current_week_spend
wow_triggered = savedAmount > ¥100 AND savedAmount > last_4_week_avg × 20%
```

| 参数 | 定义 |
|------|------|
| `last_4_week_avg` | 最近4周消费均值（滚动计算）|
| `current_week_spend` | 本周消费总额 |
| `savedAmount` | 节省金额（正数=少花了）|
| 兜底 | 不足4周数据时取默认值 |
