## 目录结构

```
docs/REVIEW/
├── README.md                    ← 本文件
├── {日期}-{功能名}-autoplan.md  ← 三方联合评审报告
├── {日期}-{功能名}-ceo-review.md
├── {日期}-{功能名}-eng-review.md
└── {日期}-{功能名}-design-review.md
```

## 命名规范

```
{YYYY-MM-DD}-{功能名}-{评审类型}.md
```

评审类型：
- `autoplan` — 三方联合评审
- `ceo-review` — CEO/Founder 评审
- `eng-review` — 工程评审
- `design-review` — 设计评审
