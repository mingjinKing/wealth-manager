# 🎨 知财 · UI/UX 设计规范

> 版本：v1.0 | 创建：2026-04-10

---

## 一、设计原则

| 原则 | 说明 | 优先级 |
|------|------|--------|
| **清晰** | 信息层次分明，一眼能看清重点 | ⭐⭐⭐⭐⭐ |
| **简洁** | 不必要的元素一律去掉 | ⭐⭐⭐⭐⭐ |
| **一致** | 同样的交互，同样的反馈 | ⭐⭐⭐⭐ |
| **温暖** | 不冰冷，像朋友一样 | ⭐⭐⭐ |

## 二、设计令牌（Design Tokens）

### 2.1 颜色系统

```kotlin
// 主色调
Primary       = 0xFFffc880  // 主题金
PrimaryDark   = 0xFF0c0c1f  // 深色（用于文字/对比）

// 背景色
Background    = 0xFFF0F0F0  // 浅灰底色
Surface       = 0xFFFFFFFF  // 纯白卡片

// 语义色
Safe         = 0xFF4A90D9  // 安全/收入
Warning      = 0xFFE55B5B  // 警告/支出

// 文字色
TextPrimary  = 0xFF1A1A2E  // 主要文字
TextSecondary= 0xFF9E9E9E  // 次要文字

// 边框
Divider      = 0xFFE8E8E8

```

### 2.2 间距系统（8pt 网格）

```kotlin
// 基础间距

// 卡片
card_padding      = 16.dp
card_radius      = 16.dp
card_elevation   = 2.dp

// 按钮
button_height    = 48.dp
button_radius    = 24.dp
button_padding   = 16.dp
```

### 2.3 字体系统

```kotlin
// 使用 Material3 Typography
```

## 三、组件规范（Components）

### 3.1 记账卡片（ExpenseCard）

```
┌────────────────────────────────────┐
│  🍗  餐饮                    ¥128  │
│       2024-01-15 12:30          │
│       备注：午餐                  │
└────────────────────────────────────┘

规范：
- 左侧 emoji + 分类名（16sp）
- 右侧金额（18sp, Bold）
- 下方日期 + 备注（12sp, Secondary）
- 内边距：16dp
- 圆角：12dp
```

### 3.2 金额展示（AmountText）

```kotlin
// 支出金额 - 红色
Text(
    color = Warning,
    fontWeight = Bold,
    fontSize = 18.sp
)

// 收入金额 - 绿色
Text(
    color = Income,
    fontWeight = Bold,
    fontSize = 18.sp
)

// 总资产 - 深色
Text(
    color = TextPrimary,
    fontWeight = Bold,
    fontSize = 24.sp
)
```

### 3.3 按钮样式

```kotlin
// 主按钮（记账 FAB）
height = 56.dp
cornerRadius = 28.dp  // 圆形
background = Primary

// 次按钮
height = 44.dp
cornerRadius = 22.dp
outline = Outline

// 文字按钮
无背景，无边框
color = Primary
```

### 3.4 底部导航（BottomNav）

```
┌─────────────────────────────────────────┐
│   🏠        💡        💡        ⭐      │
│  记记账    算算账    怎么花    攒点钱    │
│   ●          ○         ○         ○     │  ← 选中态为 Primary 实心
└─────────────────────────────────────────┘

规范：
- 图标大小：24dp
- 标签字号：12sp
- 选中颜色：Primary
- 未选中颜色：TextSecondary
- 高度：80dp（含安全区域）
```
