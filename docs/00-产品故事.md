> 版本：v1.0 | 创建：2026-04-10

## 一、用户画像（User Persona）

### 1.1 核心用户：小明

| 属性 | 描述 |
|------|------|
| **姓名** | 小明 |
| **年龄** | 28岁 |
| **职业** | 互联网产品经理，月入 2万 |
| **城市** | 北上广深 |
| **财务状态** | 有存款但不多，想理财却不懂投资 |
| **痛点** | 每月工资花光，不知道钱花哪了 |

### 1.2 用户特征

```
✓ 记账断断续续，用过钱迹、随手记
✓ 知道要省钱，但坚持不下来
✓ 对"理财"有需求但怕踩坑
✓ 更在乎"不乱花钱"而非"投资收益"
✓ 愿意为实用工具付费
```

### 1.3 典型场景

```
小明每月工资 2 万，但总是月光。
他想知道：
- 我的钱都花哪了？
- 哪些钱其实可以不花？
- 下个月怎么能少花点？

他需要的是：消费结构透视 + 切实可行的省钱建议
```

## 三、核心场景（Core Scenarios）

```
【用户】

【知财】
"你本月在餐饮上花了 ¥3200，比上月多 28%，

 建议：尝试自己做饭，每周能省 ¥100+"

【用户】
"哦，原来外卖吃了这么多..."
```

### 场景2：想买大件

```
【用户】
"我想买台相机，预算 1 万"

【知财】
"结合你的消费习惯和财务状况：
 - 1万占你月收入的 50%，偏高
 - 你近3个月有 ¥2000 的电子产品超支
 - 历史数据显示相机价格在 11.11 有折扣
建议：等 11.11，或者先买二手过渡"
```

### 场景3：哇时刻

【知财】
⭐ 哇时刻触发！

 主要是因为减少了冲动外卖。
 继续保持！"

## 四、产品愿景（Product Vision）

### 4.1 一句话愿景

> **知财 — 帮你成为更聪明的消费者**

### 4.2 三句话故事

```
每天3秒记账，
每周一次消费透视，
每月省下一笔"冤枉钱"。

知财不是告诉你"不该花钱"，
而是帮你看清"钱花得值不值"。
```

### 4.3 差异化定位

| 竞品 | 告诉你的 | 知财告诉你的 |
|------|----------|-------------|
| 钱迹 | 花了多少 | **花得值不值** |
| 随手记 | 账目明细 | **哪些其实可以不花** |
| 支付宝记账 | 支出汇总 | **消费结构优化空间** |

### 4.4 品牌调性

| 维度 | 调性 |
|------|------|
| **语气** | 专业但不冰冷，像朋友一样说话 |
| **态度** | 不批判你的消费，只帮你分析 |
| **价值** | 让你省钱，不是让你抠门 |
| **视觉** | 简洁、透明、温暖 |
