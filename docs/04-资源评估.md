# 💰 消费透视镜 · 资源评估

> 整理：玄枢·麒麟 | 最后更新：2026-03-30

---

## 一、三阶段关键结论

```
原型阶段：    几乎 0 额外成本  ←  我们现在在这里
产品化阶段：  350~1800元/年   ←  100~5000用户临界
推广阶段：    7300~30000元/年 + 推广预算  ←  5000+用户
```

**最大杠杆点**：消费透视镜的核心壁垒是 **AI 透视能力**，而非服务器或流量。

---

## 二、阶段一：原型验证（0 → ~100用户）

**时间投入**：开发一个可用的 Android MVP，预计 **2~4周全职**（用 Claude Code 辅助开发）

**部署方案**

- **安卓原生应用**，核心逻辑本地跑，不需要服务器
- AI 分析用**现有 API**（MiniMax / 阿里百炼 / DeepSeek），现有算力余额完全 cover
- 用户数据存本地 SQLite，靠应用自身更新迭代，无需后端

**成本估算**

| 资源 | 原型阶段成本 |
|------|-------------|
| 服务器 | **0**（本地 APK 分发）|
| AI API | **0**（用现有余额）|
| 开发工具 | **0**（现有设备 + Claude Code）|
| 应用分发 | **0**（蒲公英 / QQ 群 / 私信分发）|

---

## 三、阶段二：产品化（100 → ~5000用户）

**触发信号**：自己用得好、有人主动要安装

**部署方案**

- 方案 A（推荐）：轻量服务器——继续用**现有阿里云（2C2G）** 续费约 99/年，搭 Node.js/Python 后端，用 LeanCloud/Bmob 等 BaaS 降低开发成本
- 方案 B：腾讯云（已有 4C8G）

**成本估算**

| 资源 | 产品化阶段成本 |
|------|---------------|
| 服务器/云资源 | **~100~300元/年** |
| AI API | **~200~900元/年**（按量，5000用户规模）|
| BaaS 服务 | **0~600元/年**（LeanCloud 免费档）|
| 域名 | **~50元/年**（可选）|
| **合计** | **约 350~1800元/年** |

---

## 四、阶段三：推广扩张（5000+ 用户）

**运维成本上升**

- 推荐配置：**4C8G** 或更高，费用 **2000~5000元/年**
- 需要 **监控、日志、备份、CDN** 等基础设施，额外预算 **500~1500元/年**

**AI 规模化成本**

- 5万用户 × 每月30次 = **150万次调用/月**
- 建议考虑**本地部署小模型**（如 Qwen2.5-7B），在服务器上跑

**合规成本**

- ICP 许可证：约 0元（个人可申请）
- 安全评估报告（等保）：**3000~8000元**（用户过万后可能需要）
- 隐私政策 + 用户协议：约 **2000~5000元**

**成本汇总**

| 资源 | 推广阶段成本 |
|------|-------------|
| 服务器升级 | **2000~5000元/年** |
| 运维 + CDN | **500~1500元/年** |
| AI 规模化 | **1800~9000元/年**（或自建小模型）|
| 合规 | **3000~15000元/年** |
| 推广获客 | **待定（看预算）** |
| **合计** | **~7300~30000元/年 + 推广费** |
